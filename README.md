🚀 Nur Akdağ | Personal Portfolio

My personal portfolio project, where I bring 15 years of interior architecture discipline into the digital world, developed with modern web technologies.

🏗️ About the Project

This website is built to demonstrate how I translate my experience in physical space design into the world of pixels and code. It aims to provide users with an intuitive and comfortable experience, just like in a well-designed room.

🛠️ Technical Specifications

React: Dynamic and component-based user interface.

Tailwind CSS: Modern and responsive design.

Multilingual Support: Content management in both English and Turkish.

Dark Mode: Eye-friendly night mode integration.

Axios: Secure data communication with external services (POST requests).

🌟 Highlights

Responsive Design: Full compatibility across mobile, tablet, and desktop devices.

Clean Code: "Separation of Concerns" principle using isolated data (data.js) and component structure.
